# 🌸 Bloom

> 소상공인의 인스타가 피어나는 곳

Bloom은 자영업자와 소상공인을 위한 Instagram 마케팅 자동화 SaaS입니다.  
AI 캡션 생성, 예약 발행, 성과 분석을 한 곳에서 — 1인 매장도 전문가처럼.

## ✨ Features

- 🏠 **대시보드** — 팔로워·좋아요·댓글을 한눈에, Bento 그리드 레이아웃
- ✨ **새 게시물** — AI 캡션 자동 생성 (3가지 톤 + 6가지 카테고리)
- 📅 **예약 발행** — 캘린더 기반 일정 관리 + 모바일 미리보기
- 📊 **성과 분석** — Chart.js 기반 주간/요일별 시각화 + AI 인사이트
- ⚙️ **설정** — 매장 프로필, 목표, 6가지 테마, 플랜 관리

## 🎨 Design System

- **Brand**: Bloom 🌸 (소문자 워드마크)
- **Colors**: Pink · Purple · Peach · Lavender · Mint · Yellow 시그니처 팔레트
- **Gradients**: Sunset · Dreamy · Spring · Rainbow · Dark
- **Typography**: Pretendard (한국어 친화 가변 폰트)
- **Layout**: 272px 사이드바 + Bento 그리드 메인

## 🚀 Tech

- 순수 HTML/CSS/JS (프레임워크 없음)
- Chart.js for 데이터 시각화
- Pretendard via jsDelivr CDN
- 완전 반응형 (Mobile-first)

## 📦 Deploy

이 리포지토리는 Vercel에 자동 배포됩니다. 단일 `index.html` 정적 사이트.

```bash
# 로컬 미리보기
python3 -m http.server 8000
# 또는
npx serve .
```
